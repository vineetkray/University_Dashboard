# University_Dashboard

# 🎓 University Analytics Dashboard (Power BI)

A comprehensive University Analytics Dashboard built using Microsoft Power BI to analyze student performance, attendance, placements, academic performance, fee collection, and department-wise insights.

---

## 📌 Project Overview

This dashboard helps university administrators analyze student data through interactive visualizations.

The project contains four dashboard pages:

- University Overview
- Student Analytics
- Finance & Placement Analysis
- Average Marks Analysis

---

## 📊 Dashboard Pages

### 1️⃣ University Analytics Dashboard

Features:

- Total Students KPI
- Average Attendance
- Average CGPA
- Total Departments
- Total Courses
- Admission Year Trend
- Gender Distribution
- Category Distribution
- Department-wise Student Count

---

### 2️⃣ Student Analytics

Features:

- Gender Filter
- Department Filter
- Students by State
- Grade Distribution
- Department-wise Students
- Pass vs Fail Analysis

---

### 3️⃣ Finance & Placement Dashboard

Features:

- Fee Paid vs Pending
- Placement Rate
- Collected Fees
- Average Package by Company
- Department-wise Placement Rate

---

### 4️⃣ Average Marks Dashboard

Features:

- Department Filter
- Subject Filter
- Semester Filter
- Gender Filter
- Decomposition Tree
- Student CGPA Analysis

---
